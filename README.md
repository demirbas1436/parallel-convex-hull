# Parallel Convex Hull 🔷

Paralel programlama dersi projesi - Thread kullanarak nokta kümesinin convex hull'unu hesaplama.

## 🚀 Özellikler

- **Graham Scan Algoritması** - O(n log n) karmaşıklık
- **Paralel İşlem** - ThreadPoolExecutor ile multi-threading
- **Görselleştirme** - Matplotlib ile hull çizimi
- **Benchmark** - Seri vs paralel performans karşılaştırması
